# 🎧 True 8D & Reels Telegram Bot

Telegram-бот для создания 8D-аудио с реалистичным эффектом пространства и автоматической генерации вертикальных видео (Reels/TikTok/Shorts) с аудио-эквалайзером.

## ✨ Возможности

* **🎧 Настоящий 8D звук:** Использование панорамирования (L/R), динамического фильтра низких частот (психоакустика) и эффекта реверберации комнаты (Room Ambience).
* **⚙️ Выбор скорости:** Пользователь может выбрать скорость "вращения" звука (Медленно, Стандарт, Ураган).
* **🎬 Генерация Reels:** Автоматическое создание вертикального видео (9:16) с наложенным на фоновое изображение красивым эквалайзером (спектрумом).
* **🌍 Многоязычность:** Поддержка Русского, Английского и Узбекского языков (сохраняется для каждого пользователя).
* **⏳ Фоновые очереди:** Тяжелые процессы (рендеринг аудио и видео) вынесены в отдельные потоки (`threading` + `queue`). Бот не зависает и продолжает отвечать другим пользователям.
* **📊 Админ-панель:** Просмотр статистики использования бота командой `/admin`.

---

## 🛠 Требования

Для работы бота на сервере или локальном ПК должны быть установлены:
1. **Python 3.8+**
2. **FFmpeg** (ОБЯЗАТЕЛЬНО! Без него не будет работать ни обработка звука, ни генерация видео).
   * *Linux (Ubuntu/Debian):* `sudo apt install ffmpeg`
   * *Windows:* Скачать с официального сайта и добавить в переменные среды (PATH).
   * *macOS:* `brew install ffmpeg`

## 📦 Установка

1. Склонируйте репозиторий или скачайте код бота:
   ```bash
   git clone [https://github.com/yourusername/8d-reels-bot.git](https://github.com/yourusername/8d-reels-bot.git)
   cd 8d-reels-bot

```

2. Установите необходимые Python-библиотеки:
```bash
pip install pyTelegramBotAPI pydub

```


3. Подготовьте фоновое изображение для видео:
Поместите вертикальную картинку (рекомендуемое разрешение **1080x1920**) в папку с ботом и назовите её `my_i.jpg`.

---

## ⚙️ Настройка

Откройте файл с кодом бота и отредактируйте следующие переменные в блоке `ОСНОВНЫЕ НАСТРОЙКИ БОТА`:

```python
TOKEN = 'ТВОЙ_ТОКЕН_ОТ_BOTFATHER'  # Токен вашего бота
ADMIN_ID = 123456789              # Ваш Telegram ID (для команды /admin)
CHANNEL_ID = -100XXXXXXXXXX       # ID канала, куда бот будет дублировать готовые работы
PHOTO_FILE = "my_i.jpg"           # Имя файла с фоном для Reels
VIDEO_CAPTION = "🔥 @твоя_ссылка #reels" # Подпись к готовому видео

```

### Настройка визуала эквалайзера (по желанию):

В коде есть параметры для настройки внешнего вида звуковых волн на видео:

* `EQ_BOTTOM_OFFSET`: Отступ эквалайзера снизу (в пикселях).
* `EQ_HEIGHT`: Высота столбиков эквалайзера.
* `EQ_COLOR`: Цвет эквалайзера (в HEX формате, например "0xE6F8FF").
* `EQ_OPACITY`: Прозрачность (от 0.1 до 1.0).

---

## 🚀 Запуск

Запустите бота командой:

```bash
python bot.py

```

*Рекомендуется запускать бота на сервере (VPS) через `screen`, `tmux` или `systemd`, чтобы он работал 24/7.*

---

## 📱 Команды бота

* `/start` — Запустить бота и выбрать язык.
* `/lang` — Изменить язык интерфейса.
* `/admin` — Показать статистику (доступно только пользователю с `ADMIN_ID`).
* *(Отправить аудиофайл)* — Начать процесс создания 8D трека.

## 📝 Примечания

* Ограничение Telegram на отправку/получение файлов через стандартный Bot API — **20 МБ**. В коде стоит защита от загрузки файлов большего размера.
